# 🏨 Hotel Management System

✨ Professional PHP-Based Hotel Booking Application

A simple yet effective Hotel Management System developed using PHP, HTML, and CSS. The application allows users to book hotel rooms, select room categories, calculate booking costs automatically, and generate an instant booking receipt.

---

## 📌 Highlights

- Dynamic hotel room booking system
- Automatic bill calculation based on room type and duration
- Multiple room categories with different pricing
- Instant booking receipt generation
- Responsive and modern user interface
- Beginner-friendly PHP project
- Easy to deploy on XAMPP server

---

## 🚀 Features

### Customer Booking

- Enter customer name
- Select preferred room type
- Enter number of booking days
- Submit booking details

### Room Categories

#### Standard Room
- Price: ₹1000 per day

#### Deluxe Room
- Price: ₹2000 per day

#### Suite Room
- Price: ₹3500 per day

### Bill Calculation

- Calculates total bill automatically
- Displays room charges
- Shows booking summary
- Generates final receipt

### Booking Receipt

- Customer information
- Room details
- Number of days
- Price per day
- Total bill amount
- Booking confirmation message
